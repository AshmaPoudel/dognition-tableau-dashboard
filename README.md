# Optimizing the Free-to-Paid Journey
## A Behavioral Retention Analysis of Dognition

## Business Problem
Dognition operates on a freemium model where users can complete up to 7 games before hitting a paywall.
This project evaluates whether the current free-to-paid structure effectively drives engagement and long-term retention.

## Key Findings
- Significant retention drop after Game 20 (paywall trigger)
- Participation declines sharply after Game 20
- Shorter inter-test intervals strongly correlate with completion
- Growth spikes are promotion-driven and not sustainable

## Strategic Recommendations
- Extend free access to 25 games
- Introduce streak-based gamification before paywall
- Trigger upgrade prompts during peak engagement hours
- Implement referral incentives to stabilize acquisition
